More Information

How do I use this?

If you already know the ICD9 or ICD10 code that you want to use, just type it in and hit "Enter."

If you want to search for a code, just start by typing the term (e.g., "knee") and then choose from the list of matching codes. When you are ready, click "Crosswalk Code" to show matches.

In some cases, there is an exact match. However, in many cases there are ambiguities in translation that will require knowledge about the original case to make an appropriate translation.

What is your data source?

The crosswalk is freely available from the Centers for Medicaid and Medicare Services, 2012 ICD-10-CM and GEMs. Some sites are quite willing to sell you this data second-hand.

We've taken this freely available crosswalk and built a very simple system for visualizing the crosswalkable codes.

Glossary

All glossary explanations are taken from the General Equivalence Mappings, Documentation for Technical Users [PDF].

Approximate Match
Indicates complete meaning of the source and the target are not considered equivalent.

Exact Match
Indicates the complete meaning of the source and the target are considered equivalent and the source and target translate only to each other in both GEMs files.

Combination
Indicates that a cluster of target system codes—consisting of two, three, four, or more codes— taken together provide an adequate translation of the source system code.

Choice 1
Indicates that there is a pick-list of choices that work together. Each instance of a complete translation is referred to as a "cluster."


What are ICD9/ICD10 codes?

If you're not sure, this site probably isn't helpful. ICD is an abbreviation for International Statistical Classification of Diseases and Related Health Problems. This is a billing coding system that applies to hospital visits.
